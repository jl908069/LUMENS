# LUMENS

## Annotation Interface for [LUMENS](https://jl908069.github.io/LUMENS/)
```
LUMENS/
├── index.html
├── annotator.js
├── annotator.css
├── lib
├── tokens/
│   ├── fiction_001.tokens.tsv #BookNLP output with full tokens
│   ├── fiction_002.tokens.tsv
│   └── fiction_003.tokens.tsv
└── annotations/
    ├── fiction_001.tsv #Pre-annotated tsv
    ├── fiction_002.tsv
    └── fiction_003.tsv
```

## Export
Please copy and paste the exported tsv file into any text editor. Name it as {book_id}.tsv (the same as your input tsv file name).
